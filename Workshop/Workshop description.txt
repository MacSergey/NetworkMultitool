[code][h1]Overview[/h1]
[list][h2]This mod provides many different tools for working with networks[/h2][/list]
[/code]

[h1]What’s Included[/h1]
[list] 
[*] Add node mode: add node to exist segment
[*] Remove node mode: remove node and join segments
[*] Union nodes mode: union two nodes in one
[*] Intersect segments mode: make node in segment intersection point
[*] Align in line mode: Changes the position of the nodes so that they line up in one smooth line
[*] Slope nodes mode: Changes the height of the nodes so that the slope between them becomes the same value
[*] Create loop mode: Creates a loop that connects two segments with an even curve (straight - curve - straight)
[*] Create connection mode: Creates a connection for any two segments with even curves (curve - straight - curve)
[/list]

[img]https://i.imgur.com/8Ff2swh.png[/img]

Available translations: Chinese, Finnish, French, German, Japanese, Polish, Portuguese, Russian, Spanish
